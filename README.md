# Fake News Predictor

MVP da Sprint 4 (Qualidade de Software, Segurança e Sistemas Inteligentes) da Pós-Graduação em Engenharia de Software PUC-RIO

## ℹ️ Descrição

Fake News Predictor é uma aplicação capaz de predizer se notícias são verdadeiras ou falsas. Ela funciona com notícias de língua inglesa e foi construída com um modelo de classificação de machine learning que possui uma acurácia de 77%.

## 🛠️ Instalação

1. Clone o repositório:<br>
    `https://github.com/carlosedcec/fake-news-predictor`

2. Instale as dependências da API:<br>
    Acesse a pasta "api" e rode o comando `pip install -r requirements.txt`

3. Rode o servidor:<br>
    Ainda na pasta "api" rode o comando `python app.py`

4. Acesse a aplicação em:<br>
    `localhost:5000`
